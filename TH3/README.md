# Website NodeJS Express EJS

## Giới thiệu

Đây là bài thực hành xây dựng website bằng NodeJS, Express và EJS.

Website gồm 4 trang:

- Trang chủ
- Trang danh sách
- Trang chi tiết
- Trang liên hệ

## Công nghệ sử dụng

- NodeJS
- ExpressJS
- EJS
- CSS

## Chức năng

- Layout dùng chung
- Menu điều hướng
- Vòng lặp hiển thị danh sách
- If/else hiển thị trạng thái
- Route động /detail/:id

## Cách chạy project

Cài thư viện:
npm install

Chạy server:
node app.js

Mở trình duyệt:
http://localhost:3000
